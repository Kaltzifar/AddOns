Welcome to ElvUI Enhanced.

This version of ElvUI is a highly optimized version of the original ElvUI that uses less CPU cycles and add-on memory.

Here is a complete list of all the changes made to the original ElvUI:

  • Automatic role detection for battleground and raid groups.
  • Auto release in battleground after player dies.
  • Automatic changing of equipment set when changing specialization or entering a battleground.
  • GPS Locator for target and focus frame, showing direction and range to party or raid member.
  • Target Range data text module displaying the range to your currently selected target.
  • Eclipse prediction on eclipse bar for Balance Druids.
  • Automatic selection of highest vendor value quest rewards (you can still pick another if you like).
  • Automatic changing of the watched faction on your reputation bar to the faction you got reputation points for.

The following changes have been mergend back into the original ElvUI:

  • Numerous LUA Optimizations reducing CPU usage by 25% or more compared to the original ElvUI.
  • Numerous LUA Optimizations reducing memory usage by 75% or more over time compared to the original ElvUI.
